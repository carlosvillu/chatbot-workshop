# Chatbot Workshop

## prerrequisitos

- Cuenta en OpenAI
    - Creditos para usar la API

- Cuenta en Pinecone
    - Crear un Indice

## Instalación

```sh
git clone https://github.com/carlosvillu/chatbot-workshop
```

```sh
cd chatbot-workshop
npm run phoenix 
```
